# CyberGame（五章完结版）

电子取证剧情向网页游戏，当前已实现完整五章最小可玩流程：

- 第一章：截图完整性与证据判断
- 第二章：时间线还原与可行性推理
- 第三章：便签解密与线索溯源
- 第四章：交叉验证与误导识别
- 第五章：结案报告与责任收束

## 本地运行

```bash
npm install
npm run dev
```

默认地址：`http://localhost:5173`

## 测试与构建

```bash
npm run test
npm run build
```

## 当前能力

- 剧情模式 / 侦查模式切换
- 地点探索与线索采集
- 线索状态机（未发现/已发现/已解析/已用于推理）
- 章节门控（Gate A/B/C）
- 五章完整配置驱动流程
- 第二章时间线面板
- 第四/五章分级失败提示
- 章节自动推进（CH1 -> CH5）
- 终章结案摘要面板与“重新开始调查”
