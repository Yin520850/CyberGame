# CyberGame（前两章原型）

电子取证剧情向网页游戏原型，当前实现到前两章：

- 第一章：截图完整性与证据判断
- 第二章：时间线还原与可行性推理

## 本地运行

```bash
npm install
npm run dev
```

默认地址：`http://localhost:5173`

## 测试与构建

```bash
npm run test
npm run build
```

## 当前能力

- 剧情模式 / 侦查模式切换
- 地点探索与线索采集
- 线索状态机（未发现/已发现/已解析/已用于推理）
- 章节门控（Gate A/B/C）
- 第一章证据判断
- 第二章时间线面板与可行性判断
- 第二章通关后第三章入口提示
